PicMetric API Doc's

Register a user

GET

ENDPOINT: /api/auth/register/

Request


    		  "username": "username",
    		  "password": "password"

Response


				"id": 1
				"username": "username"

Login a user

POST

ENDPOINT: /api/auth/login/

Request


        "username": "username"
        "password": "password"

Response


        "id": 1,
        "username": "username"

Logout a user

GET

ENDPOINT: /api/auth/logout/

Response


        "message": "Logged out successfully."
